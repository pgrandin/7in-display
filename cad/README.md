# MPI7002 CAD Model

CAD reference model for the LCDwiki `MPI7002` / `7inch HDMI Display-C`
module (also sold rebranded, e.g. Hosyond 7" — Amazon `B09XKC53NH`; the
EDID and USB descriptors identify the same QDTECH MPI7002 board).

## Files

- `MPI7002.step` / `MPI7002.stp` - STEP B-rep assembly (same file, two
  extensions). Use this for CAD work.
- `MPI7002.3mf` - 3MF mesh export of the same geometry
- `MPI7002.scad` - editable OpenSCAD source model
- `MPI7002_github_preview.stl` - GitHub-renderable preview mesh
- `MPI7002_front_panel.step` / `.stp` / `.3mf` - 3D-printable front panel
  (bezel); print the 3MF face-down, no supports
- `MPI7002_front_panel_github_preview.stl` - panel preview mesh
- `MPI7002_panel_assembly.step` - display module + panel combined, for
  CAD reference
- `MPI7002_case_back.step` / `.stp` / `.3mf` - 3D-printable back case
  that pairs with the front panel; print the 3MF back-face-down
- `MPI7002_case_back_github_preview.stl` - case preview mesh
- `MPI7002_case_assembly.step` - display module + panel + case combined

## Front Panel

Generated by `scripts/generate_panel.py`. The display hangs behind the
panel, attached through its four mounting-ear holes:

- Plate `174.90 x 134.27 x 3.00 mm` (5 mm border around the PCB), outer
  corners R6, front edges chamfered 1.5 mm at 45 degrees.
- Window opens the LCD active area plus 1.0 mm per side
  (`156.21 x 87.92 mm`); the bezel overlaps the touch lens by at least
  2.3 mm all around, so the lens seats flat on the panel back.
- Four Ø8 bosses on the back drop down to the PCB mounting ears. Drive
  `M3 x 8` screws from behind through the ear holes into the Ø2.8 pilot
  holes (thread-forming in PLA/PETG). For heat-set inserts, change
  `PILOT_D` to 4.6 in `scripts/generate_panel.py` and regenerate.
- Boss faces sit 0.18 mm short of the nominal PCB plane so the screws
  lightly preload the glass against the panel; the module's foam tape
  layers absorb it.
- Print face-down (front face on the bed): flat plate, straight bosses,
  45-degree chamfers — no supports needed.

Regenerate with:

```bash
.venv-cad/bin/python scripts/generate_panel.py
```

## Back Case (with Raspberry Pi 5 bay)

Generated by `scripts/generate_case.py`. Completes the enclosure around
the front panel and carries a Raspberry Pi 5 with fully internal cabling:

- Exterior `202.90 x 134.27`, back at z -30.50 (overall depth with panel:
  42.38 mm). The extra width is an internal cable bay on the right; the
  front panel is unchanged and its right border rests on the bay roof.
- The display's HDMI, micro-USB, and backlight switch open into the bay
  (30 mm deep). Use right-angle or short-boot plugs; straight full-size
  HDMI plugs will not fit. No display ports are exposed outside.
- Pi 5 mounts component-side-up on four Ø6 standoffs (M2.5 pilot holes
  Ø2.05, or 3.5 for heat-set inserts; board footprint x 100..156,
  y 10..95). Its USB-C / micro-HDMI edge faces the bay for the internal
  micro-HDMI-to-HDMI and USB-A-to-micro-USB (touch) jumpers; the
  USB/Ethernet edge faces the interior and the GPIO header stays
  reachable. Opening the case is required for USB/Ethernet/SD access.
- The only external openings: a `16 x 13` cable pass-through slot in the
  bay wall (power cord, optionally Ethernet) and nine vent slots in the
  back under the Pi.
- Four Ø9 hollow pillars rise from the case floor to the PCB mounting
  ears. `M3 x 35` socket-head screws from the back seat in Ø6.5
  counterbores and thread into the front panel's bosses — four screws
  clamp case, display, and panel. (With the case, skip the panel-only
  `M3 x 8` screws.)
- Print back-face-down: walls, pillars, and standoffs rise straight up.
  The bay roof is a single ~33 mm bridge — fine with good part cooling,
  or add supports just under it.
- `MPI7002_case_assembly.step` includes an approximate Pi 5 envelope
  (board, port stacks, active-cooler and GPIO keepouts) used for the
  clearance checks; connector positions on it are approximate.

Regenerate with:

```bash
.venv-cad/bin/python scripts/generate_case.py
```

## Dimensions Used

- PCB: `164.90 mm` wide x `1.60 mm` thick. The main body is `106.96 mm`
  tall; four `8.00 mm` wide mounting ears at the corners extend it to
  `124.27 mm` overall (`8.655 mm` edge recess between the ears,
  `148.90 mm` between ear inner edges). All outline corners are
  `R4.00 mm`, making the ear tips semicircular.
- Total stack thickness: `8.88 mm` (PCB 1.60 + LCD back tape 2.00 +
  LCD 3.50 + CTP back tape 0.30 + CTP lens 1.48)
- Overall envelope including HDMI connector: `164.90 x 124.27 x 15.15 mm`
- Mounting holes: `4x Ø3.00 mm`, spacing `156.90 x 114.96 mm`
  (4.00 mm from side edges, 4.655 mm from top/bottom edges)
- LCD backlight: `164.90 x 100.00 mm`, top edge 9.52 below PCB top
- CTP lens OD: `164.28 x 99.17 mm`, 0.21 from right edge, 10.08 from top
- CTP visual area: `154.68 x 87.02 mm`, 6.75 from right, 13.80 from top
- LCD active area: `154.21 x 85.92 mm`, 6.99 from right, 14.35 from top

Connectors sit on the right edge (viewed from the front), mounted on the
PCB back face and extending 6.27 mm (HDMI) behind it. Vertical centers,
measured from the PCB top edge:

- HDMI type A ("Display"): `20.25 mm`
- micro-USB ("Touch", upper): `38.62 mm`
- micro-USB ("Touch", lower): `51.73 mm`
- Backlight slide switch: `62.87 mm`

## Coordinate System

- Units are millimeters.
- Origin is the lower-left PCB corner viewed from the front (screen side).
- `+X` goes right, `+Y` up, `+Z` out of the screen toward the viewer.
- `z = 0` is the PCB back face (component side). The screen stack spans
  `z = 0 .. 8.88`; connectors span `z = -6.27 .. 0`.

## Accuracy Notes

The board outline, hole pattern, screen layer footprints, stack thickness,
and connector vertical positions are taken directly from the official
outline drawing. Note on reading that drawing: the front view dimensions
the hole pattern (`156.90 x 114.96`, holes 4.00/4.66 from the edges) while
the back view dimensions the outline (`106.96` body height, `8.65` ear
recess, `148.90` between ear inner edges) — the two views describe the
same eared outline, verified against the edge profile of the product
photo. One caveat:

- **Connector body sizes are standard, not dimensioned.** Only the HDMI
  height (6.27) comes from the drawing. HDMI body 15.0 x 11.6, micro-USB
  7.7 x 5.4 x 2.45, switch 9.0 x 4.2 x 3.6 are typical receptacle
  dimensions, modeled flush with the PCB edge (the published product size
  implies no meaningful in-plane overhang). Measure before cutting tight
  connector openings.

The thin `*_marker` / `*_reference` solids on the glass are alignment aids
for the active/visual areas; suppress them in CAD if they get in the way.

## Regenerate

The STEP, 3MF, and preview STL are generated with CadQuery/OpenCascade:

```bash
uv venv .venv-cad --python 3.11
uv pip install --python .venv-cad/bin/python -r requirements-cad.txt
.venv-cad/bin/python scripts/generate_step.py
```

Validate the STEP by reading it back through CadQuery:

```bash
.venv-cad/bin/python - <<'PY'
import cadquery as cq
shape = cq.importers.importStep("cad/MPI7002.step")
bb = shape.val().BoundingBox()
print(f"x={bb.xlen:.2f} y={bb.ylen:.2f} z={bb.zlen:.2f}")
print(f"zmin={bb.zmin:.2f} zmax={bb.zmax:.2f}")
print("solids=", len(shape.solids().vals()))
PY
```

Expected readback:

```text
x=164.90 y=124.27 z=15.15
zmin=-6.27 zmax=8.88
solids= 11
```

The `15.15 mm` Z extent matches the published product size
(`164.9 x 124.27 x 15.15 mm`): 8.88 stack plus the 6.27 HDMI connector
behind the PCB.
