# MPI7002 CAD Model

This directory contains a practical CAD reference model for the LCDwiki
`MPI7002` / `7inch HDMI Display-C` module.

## Files

- `MPI7002.scad` - editable OpenSCAD source model
- `MPI7002.step` - STEP B-rep CAD model for CAD assemblies
- `MPI7002.stp` - same STEP model with `.stp` extension
- `MPI7002_envelope.stl` - generated STL preview mesh

## Dimensions Used

Sources:

- LCDwiki `7inch HDMI Display-C`: https://www.lcdwiki.com/7inch_HDMI_Display-C
- Product dimensions image: https://www.lcdwiki.com/res/MPI7002/MPI7002_Product_Dimensions.png
- MPI7002 datasheet: https://www.lcdwiki.com/res/MPI7002/7inch_MPI7002_datasheet.pdf

- Product size: `164.90 x 124.27 x 8.88 mm`
- LCD size: `164.90 x 100.00 x 3.50 mm`
- Capacitive touch lens OD: `164.28 x 99.17 x 1.48 mm`
- Touch sensor visual area: `154.68 x 87.02 mm`
- LCD active area: `154.21 x 85.92 mm`
- LCD back tape: `2.00 mm`
- Touch back tape: `0.30 mm`
- Mount holes: `4x 3.00 mm diameter`
- Hole spacing: `148.90 x 114.96 mm`
- PCB corner radius: `4.00 mm`

## Coordinate System

- Units are millimeters.
- Origin is the lower-left PCB corner when looking at the model from the
  front.
- `+X` goes right.
- `+Y` goes toward the top edge of the display.
- `+Z` goes outward from the PCB front face.

## Accuracy Notes

This is an envelope and mounting model. The board outline, product envelope,
screen areas, stack thickness, and mounting holes are based on published
dimensions. Use `MPI7002.step` as the CAD artifact.

The connector and IC keepouts are approximate. They are included to make early
enclosure work safer, but they should be checked with calipers before designing
a tight enclosure or connector cutouts.

The STEP and OpenSCAD models have subtracted mounting holes in the PCB. The STL
marks mounting holes with small raised cylinders because STL has no native
feature semantics.

## Regenerate STEP

The STEP model is generated with CadQuery/OpenCascade:

```bash
uv venv .venv-cad --python 3.11
uv pip install --python .venv-cad/bin/python -r requirements-cad.txt
.venv-cad/bin/python scripts/generate_step.py
```

Validate the STEP by reading it back through CadQuery:

```bash
.venv-cad/bin/python - <<'PY'
import cadquery as cq
shape = cq.importers.importStep("cad/MPI7002.step")
bb = shape.val().BoundingBox()
print(f"x={bb.xlen:.2f} y={bb.ylen:.2f} z={bb.zlen:.2f}")
print(f"xmin={bb.xmin:.2f} ymin={bb.ymin:.2f} zmin={bb.zmin:.2f}")
print(f"xmax={bb.xmax:.2f} ymax={bb.ymax:.2f} zmax={bb.zmax:.2f}")
print("solids=", len(shape.solids().vals()))
PY
```

Expected STEP readback:

```text
x=171.17 y=124.27 z=8.88
xmin=-6.27 ymin=-0.00 zmin=0.00
xmax=164.90 ymax=124.27 zmax=8.88
solids= 13
```

The `171.17 mm` X extent includes the approximate HDMI connector keepout
extending `6.27 mm` left of the PCB.

## Regenerate STL Preview

OpenSCAD is not installed on the workstation this was created on, so the STL
preview is generated by a small Python mesh writer:

```bash
./scripts/generate_stl.py
```

Validate the STL preview:

```bash
assimp info cad/MPI7002_envelope.stl
```

Expected bounds:

```text
Minimum point (-6.27, 0.00, 0.00)
Maximum point (164.90, 124.27, 8.88)
```
